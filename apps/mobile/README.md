# LifeSCC Mobile App

React Native mobile application built with Expo.

## Setup

```bash
cd apps/mobile
npm install
```

## Development

```bash
# Start Expo development server
npm start

# Run on Android
npm run android

# Run on iOS  
npm run ios
```

## Structure

- `/app` - Expo Router file-based routing
- `/components` - Reusable components
- Mobile app screens mirror web app functionality
